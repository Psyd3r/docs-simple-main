## Cliente

 Esta tela do sistema é projetada para o gerenciamento de informações de clientes. Através dela, é possível:
<Steps>
 #### Realizar buscas e localizar rapidamente clientes específicos;
 #### Visualizar as principais informações de cada cliente em uma tabela organizada;

# Detalhamento das Ações

 ## Visualizar Cliente
Esta funcionalidade permite consultar as informações básicas de um cliente através de uma janela modal prática e objetiva. É uma opção de consulta rápida que não permite alterações, ideal para verificar dados sem risco de modificações acidentais.

## Editar Cliente
Permite acessar o formulário completo de cadastro do cliente para realizar alterações nas informações já registradas. Esta ferramenta é essencial para manter os dados dos clientes sempre atualizados conforme suas necessidades comerciais e operacionais.

## Visualizar Propriedades
Direciona para uma tela dedicada que lista todas as propriedades rurais vinculadas ao cliente selecionado. Esta visualização oferece uma visão completa do portfólio de propriedades do cliente, facilitando o gerenciamento territorial e acompanhamento das atividades.



A partir desta tela, é possível também cadastrar novas propriedades através do botão "Adicionar", que abre um formulário específico com recursos avançados como integração com mapas e sistema de coordenadas GPS.

## Histórico de Limite de Crédito
Apresenta um registro cronológico completo de todas as alterações realizadas nos limites de crédito do cliente. Esta ferramenta é fundamental para controle financeiro, auditoria e acompanhamento da evolução do relacionamento comercial.

## Excluir Cliente
Remove permanentemente o cliente do sistema. Esta é uma ação irreversível que requer confirmação e deve ser utilizada com cautela, preferencialmente após verificação de que não há operações importantes vinculadas ao cliente.

### Navegação e Controles
A tela oferece recursos complementares como sistema de busca, filtros, paginação e controles de visualização, proporcionando uma navegação eficiente e intuitiva para o gerenciamento da base de clientes.
